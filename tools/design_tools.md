## Whimsical

```cardlink
url: https://whimsical.com/
title: "Whimsical - Unite teams, tasks, and tools in one place"
description: "From brainstorm to launch—plan, create, manage, and communicate in one workspace. Experience the difference of calm, focused work for you and your team."
host: whimsical.com
image: https://whimsical.com/_next_public/cards/default_2025-02-06.png
```

플로우 차트, 마인드 맵 등을 만들 수 있는 툴이다.
AI를 활용하여 생성 기능도 지원 한다.

### 주요 기능

1. 마인드맵 (Mind Maps)
- 아이디어 구조화 및 시각화
- 드래그 앤 드롭으로 쉬운 노드 생성
- 자동 레이아웃 조정
- 다양한 테마와 스타일 지원

2. 플로우차트 (Flowcharts)
- 프로세스 및 워크플로우 다이어그램 작성
- 다양한 도형과 커넥터 제공
- 스마트 라인 연결
- 자동 정렬 기능

3. 와이어프레임 (Wireframes)
- UI/UX 디자인 프로토타입 제작
- 미리 정의된 컴포넌트 라이브러리
- 반응형 디자인 지원
- 실시간 협업 가능

4. 스티키노트 (Sticky Notes)
- 브레인스토밍 및 아이디어 정리
- 칸반 보드 스타일 구성
- 색상 구분 및 그룹화
- 실시간 투표 기능

### 사용 방법

1. 문서 생성
```
1. 새 문서 생성 버튼 클릭
2. 원하는 템플릿 선택 (마인드맵, 플로우차트 등)
3. 작업 공간 설정
4. 팀원 초대 (필요시)
```

2. 협업 기능
- 실시간 공동 편집
- 댓글 및 피드백
- 변경 사항 자동 저장
- 버전 히스토리 관리

3. 공유 옵션
- 링크를 통한 공유
- 권한 설정 (보기/편집)
- 다양한 포맷으로 내보내기 (PNG, PDF 등)
- 임베드 기능

### 유용한 단축키
```
- Spacebar: 손도구 (이동)
- Ctrl/Cmd + C/V: 복사/붙여넣기
- Ctrl/Cmd + Z: 실행 취소
- Ctrl/Cmd + +/-: 확대/축소
- Delete: 선택 항목 삭제
```

### 활용 사례
1. 프로젝트 기획
- 아이디어 구조화
- 작업 흐름도 작성
- 일정 계획 수립

2. UX/UI 디자인
- 사용자 플로우 설계
- 와이어프레임 제작
- 프로토타입 테스트

3. 팀 협업
- 브레인스토밍 세션
- 회의록 정리
- 피드백 수집

### 프로 팁
1. 템플릿 활용하기
- 자주 사용하는 구조는 템플릿으로 저장
- 기존 템플릿 커스터마이징

2. 효율적인 작업을 위한 팁
- 키보드 단축키 활용
- 컬러 코딩 시스템 사용
- 주기적인 저장 및 백업

3. 협업 최적화
- 명확한 네이밍 규칙 설정
- 주기적인 동기화
- 피드백 루프 구축


## ColorMagic

```cardlink
url: https://colormagic.app/
title: "AI Color Palette Generator - ColorMagic"
description: "Create beautiful color palettes with our free AI color palette generator. Generate color palettes from names, images, text, or hex codes in seconds!"
host: colormagic.app
favicon: https://colormagic.app/img/favicon.ico
image: https://colormagic.app/img/og.png
```

키워드를 기반으로 컬러 팔레트를 생성해 주는 사이트 이다.

### 주요 기능

1. 색상 추출 (Color Extraction)
- 이미지에서 자동 색상 추출
- 주요 색상 팔레트 생성
- 색상 값 자동 복사 (HEX, RGB, HSL)
- 이미지 드래그 앤 드롭 지원

2. 색상 조합 생성
- 조화로운 색상 조합 자동 생성
- 다양한 색상 규칙 적용
  - 보색
  - 유사색
  - 삼각 배색
  - 사각 배색
- 색상 밝기/채도 조절

3. 접근성 검사
- 색상 대비 검사
- WCAG 기준 준수 확인
- 색맹/색약자를 위한 시뮬레이션
- 가독성 점수 제공

### 사용 방법

1. 색상 추출하기
```
1. 이미지 업로드 또는 드래그
2. 자동 색상 추출 대기
3. 원하는 색상 클릭하여 복사
4. 필요한 색상 형식 선택 (HEX/RGB/HSL)
```

2. 색상 조합 만들기
```
1. 기본 색상 선택
2. 원하는 색상 규칙 선택
3. 생성된 조합 확인
4. 필요시 미세 조정
```

### 고급 기능

1. 프로젝트 관리
- 색상 팔레트 저장
- 프로젝트별 구분
- 팔레트 공유 기능
- 즐겨찾기 등록

2. 내보내기 옵션
- CSS 코드 생성
- Adobe 스와치 파일
- PNG/PDF 형식
- JSON 데이터

### 활용 사례

1. 웹 디자인
- 웹사이트 컬러 스키마 설정
- UI 요소 색상 선정
- 버튼/링크 색상 최적화

2. 브랜딩
- 로고 컬러 선정
- 브랜드 가이드라인 제작
- 마케팅 자료 제작

3. UI/UX 디자인
- 접근성 고려한 색상 선정
- 인터페이스 배색
- 사용자 경험 최적화

### 프로 팁

1. 효율적인 작업을 위한 팁
- 자주 사용하는 팔레트 저장
- 단축키 활용
- 색상 히스토리 활용

2. 접근성 최적화
- 대비 ratio 3:1 이상 유지
- 색맹/색약 테스트 필수
- 텍스트 가독성 확인

3. 협업시 유의사항
- 색상 코드 통일
- 네이밍 규칙 설정
- 공유 링크 활용

### 단축키
```
- Ctrl/Cmd + C: 색상 코드 복사
- Ctrl/Cmd + S: 팔레트 저장
- Spacebar: 새로운 조합 생성
- Esc: 현재 작업 취소
```
